# AI写小说助手开发指令

## 项目概述
前端使用 React + TypeScript 开发的 AI写小说助手，调用智谱 AI API 实现续写功能，通过RAG管理上下文，让角色库和世界观等来帮助生成更好的续写内容
后端使用spring-boot + mybatisplus + radis
## 开发规范
- 使用 TypeScript，确保类型安全
- 使用 Tailwind CSS 编写样式
- API Key 从环境变量读取，不要硬编码
- 错误要有友好的提示

## 功能要求
- 实现流式输出，AI 回答要逐字显示
- 小说续写能调用角色库和世界观以及压缩后的文章摘要等来续写新的一章
- 对于世界观服务
- 写作服务
- AI调度服务
- 剧情规划服务


## 注意事项
- 要处理 API 调用失败的情况
- 加载时要有明确的提示
- 要用向量数据库来帮助完成RAG
