# 小说助手 PRD

## 核心功能

### 用户功能
- 注册：用户名、邮箱、密码
- 登录：返回 JWT token
- 个人中心：查看和编辑个人信息

### AI写作功能
- 多模式写作：续写，扩写（从大纲 → 正文），压缩（长文 → 摘要）
- 长上下文记忆： 角色不崩，剧情连续
- 风格控制： 模仿某种文风(轻小说 / 网文 / 古风)，情绪调节（压抑 / 热血 / 恐怖）

### 世界观管理功能
- 角色库：姓名，性格，背景，关系网
- 世界设定：地图，阵营，势力
- 剧情时间线

### AI剧情规划
- 自动生成：三幕式结构，启程转折，爽点设计
- 可控剧情推进：让主角失败一次，安排反转，制造悬念

### 前端页面
- 写作编辑器（类似NOtion+Word,需要登录）
- 世界观面板（需要登录）
- 侧边栏 （作品管理，章节管理，世界观管理）
- 登录/注册页
